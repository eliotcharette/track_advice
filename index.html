Programming Language Suggestion Generator

Choosing your first programming language can be a difficult choice for a number of reasons:

  • The increasing amount and variety of languages to choose from
  • How difficult the language is to learn
  • How transferable the knowledge is to other languages
  • The current market demands (job opportunites) vs personal interests

This survey compares 3 of today's best programming languages, incorporating many factors and leading you to a language that best suits both your peronsonal and professional goals. The 3 languages we will be comparing are:

JavaScript

C#

Ruby

The language the best suits you is: JavaScript!
JavaScript is an extremely popular and in demand programming language. It's faster and simpler than most programming languages making it also one of the easiest to learn; though there can be some downsides on the user end as it is interpreted differently on different browsers. Overall it's a great first language to learn for almost any kind of programmer.

The language the best suits you is: C#!
C# is one of the more difficult programming languages but once you learn it, a lot of the knowledge is transferrable to other languages. It also has a popularity and demand in jobs. This would be a great first language for you if you don't mind working a bit harder for a great base language.

The language the best suits you is: Ruby!
Ruby is one of the easiest and most dynamic programming languages to learn. It has a huge community with tons of great resources and tools, though even its staunches supports will admit it is slower than most languages and its popularity is waning. This would be a great first language for anyone who wants to pick up their first language relatively easily.
